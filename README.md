# Pokemon Project

![three monsters](docs/assets/monsters.png)

## Dependencies 

* opengl
* tinyobjloader
* glfw
* glew

## Installation Instruction

### Linux

```bash
$ git clone https://github.com/Vagos/zino-engine
$ cd zino-engine && make
```
### Windows

```bash
$ git clone https://github.com/Vagos/zino-engine
$ cd zino-engine 
$ mkdir build
$ cmake ..
```
